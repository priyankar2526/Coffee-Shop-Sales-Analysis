# ☕ Coffee Shop Sales Analysis Dashboard

## 📊 Project Overview
This project presents an interactive Excel dashboard to analyze coffee shop sales data. It helps in understanding sales performance, trends, and store-wise insights.

## 🛠️ Tools Used
* Microsoft Excel
* Pivot Tables
* Charts
* Slicers

## 📈 Key Metrics
* ✔ Total Sales: 698,812.3
* ✔ Total Orders: 149,116

## 🎯 Features
* ✔ Sales overview dashboard
* ✔ Store-wise sales analysis
* ✔ Day-wise and hourly trends
* ✔ Top 5 products based on sales
* ✔ Interactive filters using slicers

## 📂 Files Included
* Coffee_Sales_Dashboard.xlsx
* dashboard.png

## 🚀 Conclusion
This dashboard helps in identifying sales patterns, top-performing products, and store performance for better decision-making.
